# 💰 Expense Tracker

A modern, full-stack personal finance tracker designed to seamlessly log, categorize, and monitor expenses. This application is built as a highly decoupled, three-tier cloud architecture with automated CI/CD pipelines.

---

## 🚀 Live Deployments & Cloud Architecture

This project is decentralized and hosted across various professional cloud ecosystems under a zero-cost infrastructure topology:

| Layer | Component | Platform | Live URL / Status |
| :--- | :--- | :--- | :--- |
| **Frontend** | React (Vite) | [Vercel](https://vercel.com/) | [Live Website Link](https://expense-tracker-gilt-gamma-13.vercel.app) |
| **Backend** | Node.js / Express | [Render](https://render.com/) | [API Endpoint Link](https://expense-tracker-nfl8.onrender.com) |
| **Database** | Managed MySQL 8.4 | [Aiven](https://aiven.io/) | Running (DigitalOcean Bangalore Node) | aiven.io/account/
| **Repository** | Version Control | [GitHub](https://github.com/) | [GitHub Repository](https://github.com/xoya19/expense-tracker.git) |
